Header and logo

Headers

There are 7+ Header styles, every single header style include the different class. you can visit the headers pages and just copy the all code of headers.and simple rename or modify the menu items.

Default light Header
<header class="top-header navbar-default navbar navbar-static-top yamm">

     ...

</header>
Dark Header

Just replace the navbar-default class with navbar-inverse

<header class="top-header navbar-inverse navbar navbar-static-top yamm">

     ...

</header>
sticky Header

Simply add the class sticky-header to make your header sticky

<header class="top-header navbar-inverse navbar  sticky-header yamm">

     ...

</header>
Transparent Header

Just add the class .header-transparent or .navbar-fixed-top

<header class="top-header navbar-inverse header-transparent navbar-fixed-top navbar yamm">

     ...

</header>

Logo

Give the logo img path into img src or simply add the text instead of logo image

<a class="navbar-brand"> <img src ="images/logo.png" alt="logo"> </a>

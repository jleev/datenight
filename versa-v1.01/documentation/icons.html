Icons

How to use icon fonts

There are 3 icons fonts set included with the pack. Font-Awesome, Et-line, and live icons

  1. Font Awesome
  2. Live icons
  3. ET-Line icons

Font Awesome

Required file

Folder - font-awesome/

<link rel="stylesheet" href="bower_components/font-awesome/css/font-awesome.min.css">

Generate the icon

<i class="fa fa-home"></i>

ET-Line icons

Required file

Folder - et-line-icons/

<link rel="stylesheet" href="et-line-icons/et-line.css">

Generate the icon

<i class="icon-phone"></i>

Live icons

Required file

Folder - live-icon/

<script src="live-icon/raphael-min.js"></script>
<script src="live-icon/livicons-1.4.min.js"></script>

Generate the icon

<span class="livicon" data-name"edit"  data-size"32"></span>
